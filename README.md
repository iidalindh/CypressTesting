# Testing CoolCommunity with Cypress

## Installation:

1. Clone repository: https://github.com/iidalindh/CypressTesting.git
2. npm i to install node_modules
3. npm i --save-dev cypress
4. Install plugin - liveserver

---

## To run cypress:

1. npm run test
